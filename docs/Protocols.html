  • KUSConnectionListener

    Lets you observe when you’re disconnected. Core SDK only - if you’re using our UI you do not need to use this.

    var cp = ConnectionProvider.shared cp.addListener(self)

    extension AppDelegate: KUSConnectionListener { func onPubNubConnectionStatusChange(status:String) { Alert.show(“**PubNub connection status changed (status)”) } }
