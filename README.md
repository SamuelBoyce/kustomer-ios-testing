## Requirements

- A [Kustomer.com](https://www.kustomer.com/) API Key
- Xcode 12.0+
- iOS 11.0+

#### API Key

The Kustomer iOS SDK requires a valid API Key with role `org.tracking`. See [Installation](https://kustomer.readme.io/docs/installation-1)
